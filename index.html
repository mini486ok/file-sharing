      사용 방법 — 업로드에는 GitHub 토큰이 필요합니다(우측 상단 🔑). 다운로드는 누구나 가능합니다. 파일은 files/ 폴더에 커밋으로 저장됩니다.

      🔑 GitHub 토큰 설정

      업로드·삭제 시 저장소에 커밋할 권한이 필요합니다. 토큰은 이 브라우저에만 저장됩니다(localStorage).

      1. GitHub → Fine-grained token 발급 페이지 ↗ 열기
      2. Repository access → Only select repositories → file-sharing 선택
      3. Permissions → Contents → Read and write
      4. 발급된 토큰(github_pat_…)을 아래에 붙여넣기
